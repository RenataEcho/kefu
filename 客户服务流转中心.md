# **右豹项目用户****数据库****接入工作流程**

|                       |                                                                           |
| --------------------- | ------------------------------------------------------------------------- |
| **岗位名称**              | **核心职责**                                                                  |
| **右豹客服**              | 1.服务右豹非付费用户，进行意向分流，  <br>2.引导非意向用户完成数据录入并复核信息  <br>3.意向付费会员对接给专属顾问（对接岗待定） |
| **进群审核客服****外部官方交付群** | 1. 负责进群申请审核、备注校验；<br><br>2. 群内数据筛查、未达标移除、晋级拉群等（工作量较大，建议按项目分设多个岗位）         |
| **付费对接客服**            | 1. 服务已付费用户，完成数据录入、信息复核；<br><br>2. 添加进入“第三职业”交付体系，并在数据录入库完善飞书信息            |
| **“第三职业”审核员**         | 处理会员飞书门派更换，同步更新至“数据录入库”                                                   |

---

## 一、**非付费用户流程**

**1. 用户发起咨询**

用户在右豹APP内查看感兴趣的项目，点击“联系客服”。

系统展示右豹客服企业微信二维码，用户保存并扫码添加。

**2. 客服初步分流**

客服通过好友申请后，与用户沟通需求进行分流：

**意向付费用户** → 转接专属顾问

**无意向/观望用户** → 进入数据录入环节

**3. 数据录入与审核**

客服发送 **“数据录入库”二维码** 给用户，引导用户完成信息录入。

用户提交后，客服审核信息是否**完全正确**。  
✅ 正确 → 进入下一步  
❌ 有误 → 要求用户修正后重新提交或右豹客服进行修改

### **4. 社群导入与入群规范**

审核通过后，客服向用户发送对应项目的**飞书交付群二维码**。

**【必读】** 告知会员：进群时须备注 **“右豹编码 + 飞书注册手机号”**，否则申请不予通过。

### **5. 入群审核（免费群）**

审核员根据用户备注的**右豹编码/手机号**，在“数据录入库”中搜索查询。

查询到匹配信息 → 通过进群申请

未查询到 → 拒绝申请，并提示用户联系客服

### **6. 特殊规则：飞书好友添加限制**

**问题**：飞书每日好友添加上限。

**处理规则**：

**优先添加**：近10天内进行过“关键词申请”或“信息回填”的用户。

**暂缓添加**：近10天内未在右豹APP做任何申请或添加关键词动作的用户，暂不添加好友。

---

## **二、付费用户流程**

### **1. 账号注册协助**

对接客服与付费用户建立联系后，**首先**指导用户完成右豹APP注册。

协助用户注册成功，确保账号可正常使用。

### **2. 信息录入与截图核对**

用户注册完成后，客服发送 **“数据录入库”二维码**。

**【关键动作】** 明确告知用户**右豹编码**和**右豹ID**的位置，并要求用户在这两处**完整截图**（共两张图）。

用户提交信息后，客服根据截图核对录入内容是否正确。  
✅ 正确 → 进入下一步  
❌ 有误 → 引导用户修正后重新提交或对接客服进行修改

### **3. 交付体系开通**

核对无误后，客服在飞书 **“第三职业”交付体系** 中添加用户信息。

添加完成后，告知会员账号已开通，可正常激活登录。

### **4. 信息归档**

对接客服在 **“数据录入库”** 中补充完善用户的**飞书注册手机号码**及**飞书ID信息**，完成全流程闭环。

---

## **三、通用注意事项**

**1****.****编码规范**  
所有环节中涉及的“右豹编码”必须保持一致，作为核心身份识别字段。

**2****.****截图要求**  
付费用户流程中，务必强调截图需完整包含“右豹编码”和“右豹ID”，避免因截图不全导致审核延迟。

**3****.****信息同步**  
客服与审核员之间需保持信息同步，确保“数据录入库”实时更新，避免因数据延迟导致入群被拒。

**4.****异常处理**

若用户备注信息错误或不完整，审核员应拒绝申请并告知正确格式。

若飞书好友添加受限，按“特殊规则”执行优先顺序。
